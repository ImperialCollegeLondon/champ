About CHAMP

CHAMP v{{ APP_VERSION }}
 © 2021 Imperial College London

CHAMP is a web browser-based tool for simplifying the submission of computational tasks to High Performance Computing (HPC) infrastructure.

Avoid the need to learn about using the command line, connecting to remote system using tools like SSH or writing job submission scripts. Get the benefits of HPC processing power directly through your browser!
